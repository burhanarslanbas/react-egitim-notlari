# React Temel Kütüphane Kurulumları

## Material UI 
npm install @mui/material @emotion/react @emotion/styled

## Axios
npm install axios

## React Router DOM 
npm install react-router-dom

## Redux Toolkit
npm install @reduxjs/toolkit react-redux

## Icons
npm install react-icons 

### NOT: Neredeyse her projede kullanılan bu kütüphaneleri bu şekilde tek tek eklemek yerine proje başlangıcında otomatik kurulmasını sağlayan bir yapıyı kurmak gerekiyor.
